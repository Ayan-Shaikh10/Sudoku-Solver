Sudoku Solver Android

An Android application that validates and solves Sudoku puzzles using the Backtracking Algorithm. Built with Java and XML in Android Studio, the application provides an interactive 9×9 Sudoku grid, puzzle validation, automatic solving, and a clean dark-themed user interface.

Features

- Interactive 9×9 Sudoku board
- Easy number input through selection dialog
- Sudoku validation before solving
- Automatic puzzle solving using Backtracking Algorithm
- Real-time filled cell counter
- Result screen with solved puzzle visualization
- Highlighted generated cells for better readability
- Dark-themed modern user interface
- Error handling for invalid puzzles
- Clear and reset functionality

How It Works

1. Enter known Sudoku values into the grid.
2. Validate the puzzle.
3. Click Solve.
4. The Backtracking Algorithm calculates a valid solution.
5. The solved puzzle is displayed on a separate screen.
6. Generated values are visually differentiated from original user inputs.
